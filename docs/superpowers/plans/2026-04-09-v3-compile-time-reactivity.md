# v3 Compile-Time Reactivity Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Compile `useState` + `useButton` patterns from counter.tsx into a piu Behavior with state fields, button handlers, and targeted label mutations — all inferred automatically by rendering at multiple state snapshots and diffing.

**Architecture:** Extend the v2 compiler with three new stages: (1) intercept `useState`/`useButton` during mock renders, (2) perturb each state variable and diff the label output to build a dependency map, (3) emit a unified Behavior class with state fields + button handlers + refresh logic. The existing time-diff pipeline merges into the same Behavior.

**Tech Stack:** TypeScript, Preact (mock render only), piu (on-watch output), `tsx` script runner.

**Spec:** `docs/superpowers/specs/2026-04-09-v3-compile-time-reactivity-design.md`

### Task 3: Intercept useState calls

**Files:**

Add a `StateSlot` type and a monkey-patch on `preact/hooks`'s `useState` to record each state slot during rendering.

- [ ] **Step 1: Add state interception infrastructure**

After the existing helper functions and before the main section, add:

```typescript
// ---------------------------------------------------------------------------
// State discovery — intercept useState to record state slots
// ---------------------------------------------------------------------------

interface StateSlot {
  index: number;
  initialValue: unknown;
  setter: (value: unknown) => void;
  currentValue: unknown;
}

let stateSlots: StateSlot[] = [];
let stateInterceptActive = false;
let stateCallIndex = 0;

// Force a specific value for a state slot during perturbed renders.
// Key: slot index, Value: forced value. If absent, use real state.
let forcedStateValues: Map<number, unknown> = new Map();

import * as preactHooks from 'preact/hooks';

const realUseState = preactHooks.useState;

function interceptedUseState<T>(initialState: T | (() => T)): [T, (v: T | ((prev: T) => T)) => void] {
  const slotIdx = stateCallIndex++;
  const [value, setter] = realUseState(initialState);

  if (stateInterceptActive && slotIdx >= stateSlots.length) {
    const initVal = typeof initialState === 'function'
      ? (initialState as () => T)()
      : initialState;
    stateSlots.push({
      index: slotIdx,
      initialValue: initVal,
      setter: setter as (v: unknown) => void,
      currentValue: value,
    });
  }

  // If this slot has a forced value, return it instead of real state.
  const forced = forcedStateValues.get(slotIdx);
  if (forced !== undefined) {
    return [forced as T, setter];
  }

  return [value, setter];
}

// Install/remove the intercept
function installStateIntercept() {
  stateSlots = [];
  stateCallIndex = 0;
  stateInterceptActive = true;
  forcedStateValues = new Map();
  (preactHooks as { useState: typeof preactHooks.useState }).useState = interceptedUseState as typeof preactHooks.useState;
}

function removeStateIntercept() {
  stateInterceptActive = false;
  (preactHooks as { useState: typeof preactHooks.useState }).useState = realUseState;
}

function resetStateCallIndex() {
  stateCallIndex = 0;
}
```

- [ ] **Step 2: Add setState pattern analyzer**

After the button interception code, add:

```typescript
// ---------------------------------------------------------------------------
// setState pattern analysis — extract mutation logic from handler source
// ---------------------------------------------------------------------------

interface StateMutation {
  slotIndex: number;
  type: 'increment' | 'decrement' | 'reset';
  value: number; // increment/decrement amount, or reset target
}

function analyzeButtonHandler(source: string, numStateSlots: number): StateMutation | null {
  // Pattern: (c) => c + N or c => c + N
  const incrMatch = source.match(/\(?\w+\)?\s*=>\s*\w+\s*\+\s*(\d+)/);
  if (incrMatch) {
    return { slotIndex: 0, type: 'increment', value: parseInt(incrMatch[1]!, 10) };
  }

  // Pattern: (c) => c - N or c => c - N
  const decrMatch = source.match(/\(?\w+\)?\s*=>\s*\w+\s*-\s*(\d+)/);
  if (decrMatch) {
    return { slotIndex: 0, type: 'decrement', value: parseInt(decrMatch[1]!, 10) };
  }

  // Pattern: () => N (constant reset) — the outer setCount(0) wraps to () => setCount(0)
  // But the handler source is the OUTER function, e.g. "() => setCount(0)"
  // We look for a plain number argument.
  const resetMatch = source.match(/\(\)\s*=>\s*\w+\((\d+)\)/);
  if (resetMatch) {
    return { slotIndex: 0, type: 'reset', value: parseInt(resetMatch[1]!, 10) };
  }

  return null;
}
```

### Task 5: Wire intercepts into the render pipeline and build dependency map

**Files:**
- Modify: `scripts/compile-to-piu.ts`

Replace the current main section (the hardcoded watchface rendering pipeline) with a generalized pipeline that:
1. Installs intercepts
2. Does the baseline render
3. Perturbs each state slot and diffs labels
4. Collects time-dependent labels (existing T1/T2 logic)
5. Merges all dynamic labels into one set

- [ ] **Step 1: Rewrite the main pipeline**

Replace everything from `const origLog = console.log;` to the end of the file with the new pipeline. The new code:

1. Creates both test dates with `OrigDate` (keep existing pattern).
2. Installs state + button intercepts.
3. Does the T1 baseline render (mocked Date + intercepted useState/useButton).
4. Collects `stateSlots` and `buttonBindings` after baseline.
5. For each state slot with a number initial value, does a perturbed render (forces that slot to `initialValue + 42`), collects labels, diffs against baseline.
6. Does the T2 render for time diff (existing pattern).
7. Merges state-dependent labels (`sl` prefix) and time-dependent labels (`tl` prefix) into one `dynamicLabels` set.
8. Removes intercepts, restores Date.
9. Does the final render for the emitted snapshot.

The key new data structures after the pipeline:

```typescript
// State-dependent labels: labelIndex → { slotIndex, formatExpr }
const stateDeps = new Map<number, { slotIndex: number; formatExpr: string }>();

// For each state slot perturbed render:
for (let si = 0; si < stateSlots.length; si++) {
  const slot = stateSlots[si]!;
  if (typeof slot.initialValue !== 'number') continue;

  const perturbedValue = (slot.initialValue as number) + 42;
  forcedStateValues.set(si, perturbedValue);
  resetStateCallIndex();

  silence();
  const appPerturbed = exampleMain();
  restore();
  if (!appPerturbed) continue;

  const ctxP: EmitContext = { /* fresh context */ };
  emitNode(appPerturbed._root, ctxP, '', null);
  appPerturbed.unmount();

  forcedStateValues.clear();

  // Diff against baseline
  for (const [idx, baseText] of baselineTexts) {
    const pertText = ctxP.labelTexts.get(idx);
    if (pertText !== undefined && pertText !== baseText) {
      // This label depends on state slot si
      const formatExpr = String(perturbedValue) === pertText
        ? `"" + this.s${si}`
        : `/* unknown format */`;
      stateDeps.set(idx, { slotIndex: si, formatExpr });
    }
  }
}
```

const hasTimeDeps = labelFormats.size > 0;
const hasStateDeps = stateDeps.size > 0;
const hasButtons = buttonBindings.length > 0;
const hasBehavior = hasTimeDeps || hasStateDeps || hasButtons;

if (hasBehavior) {

    lines.push('function pad(n) { return n < 10 ? "0" + n : "" + n; }');
    lines.push(
      'const days = ["Sun","Mon","Tue","Wed","Thu","Fri","Sat"];',
      'const months = ["Jan","Feb","Mar","Apr","May","Jun","Jul","Aug","Sep","Oct","Nov","Dec"];',
    );
  }
  lines.push('');
  lines.push('class AppBehavior extends Behavior {');

  // onCreate — init state fields + capture label references
  lines.push('  onCreate(app) {');
  lines.push('    const c = app.first;');
  for (const slot of stateSlots) {
    if (typeof slot.initialValue === 'number') {
      lines.push(`    this.s${slot.index} = ${slot.initialValue};`);
    }

    lines.push(`    this.sl${idx} = c.content("sl${idx}");`);
  }
  lines.push('  }');

  lines.push('  onDisplaying(app) {');
  lines.push('    this.refresh();');

    lines.push('    app.start();');

  if (hasButtons) {

    lines.push('      watch.addEventListener("button", (e) => this.onButton(e));');
    lines.push('      watch.addEventListener("buttonClick", (e) => this.onButton(e));');
    lines.push('    }');
  }
  lines.push('  }');

  // onTimeChanged (only if time-dependent)
  if (hasTimeDeps) {
    lines.push('  onTimeChanged() { this.refresh(); }');
  }

  // onButton (only if buttons exist)
  if (hasButtons) {
    lines.push('  onButton(e) {');
    lines.push('    const name = e && e.button;');
    for (const binding of buttonBindings) {
      const mutation = analyzeButtonHandler(binding.handlerSource, stateSlots.length);
      if (mutation) {
        const guard = `name === "${binding.button}"`;
        let action: string;
        if (mutation.type === 'increment') action = `this.s${mutation.slotIndex} += ${mutation.value};`;
        else if (mutation.type === 'decrement') action = `this.s${mutation.slotIndex} -= ${mutation.value};`;
        else action = `this.s${mutation.slotIndex} = ${mutation.value};`;
        lines.push(`    if (${guard}) { ${action} this.refresh(); }`);
      } else {
        lines.push(`    // WARNING: unrecognized handler for "${binding.button}": ${binding.handlerSource}`);
      }
    }
    lines.push('  }');
  }

  // refresh — update all dynamic labels
  lines.push('  refresh() {');
  if (hasTimeDeps) {
    lines.push('    const d = new Date();');
    for (const [idx, fmt] of labelFormats) {
      lines.push(`    this.tl${idx}.string = ${emitTimeExpr(fmt)};`);
    }
  }
  for (const [idx, dep] of stateDeps) {
    lines.push(`    this.sl${idx}.string = ${dep.formatExpr};`);
  }
  lines.push('  }');

  lines.push('}');
  lines.push('');
}
